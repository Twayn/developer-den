O-нотация

O-нотация O-notation — специальное обозначение для скорости работы алгоритма в общем случае.

Зачем нужна? Определить быстроту используемого алгоритма, сравнить алгоритмы.
Понять темп роста времени работы алгоритма с увеличением числа входных данных.
О-нотация определяет худщее время работы алгоритма.
Например, поиск по массиву перебором - сложность O(n), но искомый элемент может быть первым.
n — означает количество входных данных.
O(n * log n) - с ростом n скорость алгоритма растет не быстрее чем функция n*log n.

Примеры O-нотаций и алгоритмов в порядке увеличения времени:

  • O(1) — доступ к элементу массива (константное время);
  • O(log n) — бинарный поиск (логарифмическое время);
  • O(n) — поиск перебором (линейное время);
  • O(n * log n) — быстрая сортировка;
  • O(n2) — сортировка выбором;
  • O(n!) — задача комивояжера.
