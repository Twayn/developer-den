Хэш-таблица (Hash map)

Хэш-таблица Hash-map — структура данных, состоящая из корзин и способная отображать ключи в значения.
Хэш-функция Hash-function — способ вычислить индекс в массиве корзин.
Корзина Bucket, Slot — хранилище значений, соответсвующих индексу (связный список, дерево).
Хеш функция в идеале должна равномерно распределять ключи по индексам, чтобы не было коллизий.

В среднем константное время доступа к значению по ключу.
Но оно не гарантируется если одному ключу соответсвует несколько значений.
В таком случае - дополнительный проход по связному списку O(n), дереву O(log n).

Накладные расходны при хранении, сложность реализации.
Главное приемущество скорость изменения и доступа к данным.
Имеет смысл для относительно больших объемов данных.
Для обеспечения скорости нужно управлять коллизиями.
Возможно перестроение хэша.
Сложность выбора хэш-функции - баланс между скоростью и равномерностью.
