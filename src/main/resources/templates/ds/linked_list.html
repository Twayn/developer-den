Связанный список (Linked list)

Связный список Linked list — структура данных, состоящая из узлов (данные + ссылка на соседа(ей)).
Бывает двусвязный (ссылка на соседа слева и справа) и односвязный (ссылка только на 1 соседа).
По сравнению с массивом элементы могут быть произвольно разбросаны по памяти.
Константное время добавления и удаления элемента.
Линейное время доступа по индексу.
