Массив

Структура данных Data structure — формат хранения и способ обработки данных.
Формат определяет степерь эффективности различных операций доступа и модификации.

Массив Array — структура данных, набор элементов, каждый из которых имеет хотя бы 1 индекс.
Условно — непрерывная область памяти.
Число индексов — размерность массива (одномерные, двухмерные).
Константное время доступа к элементу.
Линейное добавление и удаление из середины.
