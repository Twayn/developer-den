Ключи

Ключ Key — атрибут(ы) однозначено идентифицирующий запись.
Используется для идентификации и связи строк между отношениями.

Суперключ Superkey — атрибут(ы) отношения, удовлетворяющий требованию уникальности.
Потенциальный ключ Candidate key, Unique key — атрибуты(ы) отношения, удовлетворяющий требованиям уникальности и минимальности.
Уникальность — атрибуты ключа уникальны для всех кортежей.
Минимальность — отсутствие в ключе подмножества атрибутов уникальных для всех кортежей.

Первичный ключ Primary key — один из потенциальных ключей, выделенный по прагматическим соображениям.
К нему предъявляются два основных и два дополнительных требования.

Основные:

  • Ключ не может иметь значение NULL.
  • Ключ должен быть уникальным.

Дополнительные:

  • Ключ должен быть коротким, компактным.
  • Ключ должен быть стабильным.

Многие СУБД по ключу строят индексы, поэтому выбор ключа влияет на физическую организацию данных.

Простой ключ Simple key — состоящий из одного атрибута.
Составной ключ Concatenated key из двух или более.
Суррогатный ключ Surrogate key — искуственно введенное и генерируемое поле, которое становится первичным ключем.
Естественный Natural key — один из потенциальных ключей выбранный в качестве первичного.
Суррогатный ключ обычно вводится чтобы лучше соответствовать требованиям к первичному ключу.

Внешний ключ Foreign key — атрибут(ы) отношения, позволяющий однозначно определить запись из какого-либо отношения.
Уникальность первичных ключей и соответствие внешних обычно контролируется БД (понятие constraints).

Зачем нужен внешний ключ?

 Он используется для связи отношений в БД и поддержания целостности.

 Всего существует 3 типа связей:

  • Один к одному — одной записи соответсвует одна из другого отношения;
  • Один ко многим (многие к одному) — одной записи соответсвовует несколько из другого отношения;
  • Многие ко многим — нескольким записям соответсвует несколько из другого отношения.

 Для реализации связи многие ко многим обычно используется дополнительная таблица.

Существуют другие виды ключей:

 альтенативные, индексные, хеш-ключи, ключи сортировки, вторичные ключи, ключи шифрования и пр.
